# Asset Discovery Presentation Materials

Discovery demo.

  and shorter/longer variants

  website demo
- [`qa-prep.md`](./qa-prep.md): preparation notes for likely audience questions
- [`capture-checklist.md`](./capture-checklist.md): screenshots and backup media
  to prepare before the meeting
- [`open-source-boundaries.md`](./open-source-boundaries.md): short note for the
  site, talk, or follow-up materials
